# Challenge #10: Queries the blockchain

Script that queries the blockchain to retrieve and display the list of transactions for each of the generated account.

## Prerequisites

- Node.js
- TypeScript
- MultiversX SDK
- Wallets generated from Challenge #1

## Setup

```bash
npx tsc --init
npm install @multiversx/sdk-core @multiversx/sdk-network-providers @multiversx/sdk-wallet
```

## Usage

Run the script:
```bash
ts-node ranking.ts
